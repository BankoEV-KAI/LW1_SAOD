# LW1_SAOD -- первая ЛР по курсу САОД 

**Задание на ЛР**

Реализация структуры **стек** на основе динамического распределения памяти.

Функции, которые необходимо реализовать:
1. Проверка пустоты стека;
2. Добавление элемента в вершину стека:
	- создание нового элемента:
		- ввод одного элемента;
		- добавление n случаных элементов; 
	- выбор элемента с вершины вспомогательного стека;
3. Удаление элемента из вершины стека:
	- удаление;
	- перемещение в вершину вспомогательного стека;
4. Вывод текущего состояния стека на экран;
5. Вывод содержимого вспомогательного стека *удаленных* элементов.
